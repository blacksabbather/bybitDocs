Delete the api key of sub account. Use the api key pending to be delete to call the endpoint. Use **sub user's api key** **only**.

:::tip
The API key must have one of the below permissions in order to call this endpoint..
* sub API key: "Account Transfer"
:::

:::danger
BE CAREFUL! The API key used to call this interface will be invalid immediately.
:::

### HTTP Request
POST `/user/v3/private/delete-sub-api`

### Request Parameters
None

### Response Parameters
None

### Request Example

```http
POST /user/v3/private/delete-sub-api HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: xxxxxxxxxxxxx
X-BAPI-API-KEY: xxxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1671180557996
X-BAPI-RECV-WINDOW: 50000
Content-Type: application/json
{}
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {},
    "retExtInfo": {},
    "time": 1671180558095
}
```
