Get the delivery price for option
> **Covers: Option**

### HTTP Request
GET `/v5/market/delivery-price`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `option`|
|symbol |false |string |Symbol name|
|baseCoin |false |string |Base coin. Default: `BTC`|
|limit |false |integer |Limit for data size per page. [`1`, `200`]. Default: `50`|
|cursor |false |string |Cursor. Used for pagination|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Product type |
|list |array |Object |
|> symbol |string |Symbol name |
|> deliveryPrice |string |Delivery price |
|> deliveryTime |string |Delivery timestamp (ms) |
|nextPageCursor |string |Cursor. Used for pagination |
