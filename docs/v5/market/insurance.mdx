Query for Bybit <a href="https://www.bybit.com/data/basic/linear/insurance-fund?symbol=BTCUSDT">insurance pool</a> data (BTC/USDT/USDC etc). The data is updated every 24 hours.

### HTTP Request
GET `/v5/market/insurance`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|coin |false |string |coin. Default: return all insurance coins |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|updatedTime |string |Data updated time (ms) |
|list |array |Object |
|> coin |string |Coin |
|> balance |string |Balance |
|> value |string |USD value |
