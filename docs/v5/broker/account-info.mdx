:::info
* Use exchange broker master account to query
:::

> API rate limit: 10 req / sec

### HTTP Request
GET `/v5/broker/account-info`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|subAcctQty |string |The qty of sub account has been created |
|maxSubAcctQty |string |The max limit of sub account can be created|
|baseFeeRebateRate |Object |Base rebase fee rate|
|> spot |string |Spot trading fee rate, e.g., 10.00% |
|> derivatives |string |Derivatives trading fee rate, e.g., 10.00%|
|markupFeeRebateRate |Object |Mark up rebate fee rate|
|> spot |string |Spot trading fee rate, e.g., 10.00% |
|> derivatives |string |Derivatives trading fee rate, e.g., 10.00%|
|ts |string |System timestamp (ms) |
