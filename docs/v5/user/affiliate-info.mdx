This API is used for affiliate to get their users information

:::tip
* Use master UID only
* The api key can only have "Affiliate" permission
:::

### HTTP Request
GET `/v5/customer/info`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|UID |**true** |string |The master account uid of affiliate|
|coin |false |string |Coin name. If not passed, "USDT" by default|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|uid |string |uid|
|vipLevel |string |VIP level |
|takerVol30Day |string |Taker volume in last 30 days |
|makerVol30Day |string |Maker volume in last 30 days |
|depositAmount |string |Deposit amount in last 30 days |
|takerVolTotal |string |Total taker volume in the past year |
|makerVolTotal |string |Total maker volume in the past year |
|tradeVolTotal |string |Total trading volume in the past year |
|depositAmountTotal |string |Total deposit amount in the past year |
|totalWalletBalance |string |Total wallet balance |
|depositUpdateTime |string |The last date time of deposit |
