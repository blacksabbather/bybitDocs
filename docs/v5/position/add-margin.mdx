Turn on/off auto-add-margin for **isolated** margin position
> **Covers: USDT perpetual(Normal Account)**

### HTTP Request
POST `/v5/position/set-auto-add-margin`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |Product type. `linear` |
|symbol |**true** |string |Symbol name |
|autoAddMargin |**true** |integer |Turn on/off. `0`: off. `1`: on |
|positionIdx |false |integer |Used to identify positions in different position modes. For hedge mode position, this param is **required**<ul><li>`0`: one-way mode</li><li>`1`: hedge-mode Buy side</li><li>`2`: hedge-mode Sell side</li></ul>|

### Response Parameters
None
