Set the leverage
> **Unified account covers: USDT perpetual / USDC contract / Inverse contract**<br/>
> **Normal account covers: USDT perpetual / Inverse contract**

### HTTP Request
POST `/v5/position/set-leverage`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type <ul><li>Unified account: `linear`, `inverse`</li><li>Normal account: `linear`, `inverse`. _Please note that `category` is **not** involved with business logic_</li></ul>|
|symbol |**true** |string |Symbol name|
|buyLeverage |**true** |string |[`1`, max leverage of corresponding [risk limit](../market/risk-limit#)]. _Note: Under one-way mode, `buyLeverage` must be the same as `sellLeverage`_ |
|sellLeverage |**true** |string |[`1`, max leverage of corresponding risk limit]. _Note: Under one-way mode, `buyLeverage` must be the same as `sellLeverage`_|
