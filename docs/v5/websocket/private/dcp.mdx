Subscribe to the dcp stream to trigger DCP function.

For example, connection A subscribes "dcp", and connection B does not.
If A is still alive, but B is disconnected, then this case will not trigger DCP. However, if A is disconnected, but B is
still alive, then DCP is triggered when reach the timeWindow threshold.

**Topic:** `dcp`

### Subscribe Example
```json
{
    "op": "subscribe",
    "args": [
        "dcp"
    ]
}
```
