### HTTP 請求
GET `/v5/spot-cross-margin-trade/borrow-token`

### 請求參數
| 參數 |是否必需| 類型 | 說明|
|:----- |:-------|:-----|------ |
|coin|false|string|幣種名稱 |

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|list|array|Object|
|> coin|string |幣種名稱 |
|> borrowingPrecision|integer |借款精度 |
|> repaymentPrecision|integer |還款精度 |

<a href="/api-explorer/v5/spot-margin-normal/borrowable-data"><Button>運行實例 >></Button></a>

---

### 請求示例

```http
GET /v5/spot-cross-margin-trade/borrow-token?coin=ETH HTTP/1.1
Host: api-testnet.bybit.com
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "success",
    "result": {
        "list": [
            {
                "borrowingPrecision": 5,
                "coin": "ETH",
                "repaymentPrecision": 4
            }
        ]
    },
    "retExtInfo": {},
    "time": 1677750685331
}
```
