該接口可以查詢賬戶下的配置

:::note
第一版本是查詢保證金模式, 支持 RM 和 PM 模式查詢, 以及支持查詢帳戶的升級狀態
:::

### HTTP 請求
GET `/v5/account/info`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|[unifiedMarginStatus](../enum#unifiedmarginstatus) |integer |帳戶狀態類型|
|marginMode |string |`REGULAR_MARGIN`（常規保證金模式） `PORTFOLIO_MARGIN`（組合保證金模式） |
|updatedTime |string |賬戶數據更新的時間，毫秒時間戳|

<a href="/api-explorer/v5/account/account-info"><Button>運行實例 >></Button></a>

### 請求示例

```http
GET /v5/account/info HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672819328519
X-BAPI-RECV-WINDOW: 5000
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "marginMode": "PORTFOLIO_MARGIN",
        "updatedTime": "1672737044000",
        "unifiedMarginStatus": 3
    }
}
```
