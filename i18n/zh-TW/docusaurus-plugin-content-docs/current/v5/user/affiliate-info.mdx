該接口是給代理商查詢他們用戶的一些基本信息的

:::tip
* 僅支持使用母帳戶uid
* 若要查詢該接口, api key僅能擁有代理商權限, 若擁有任何其他權限想, 請移除
:::

### HTTP 請求
GET `/v5/customer/info`

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|UID |**true** |string |代理的母帳戶uid|
|coin |false |string |幣種名稱. 若不傳, 則默認返回USDT數據|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|uid |string |帳戶uid|
|vipLevel |string |VIP等級 |
|takerVol30Day |string |過去30天的吃單交易量 |
|makerVol30Day |string |過去30天的掛單交易量 |
|depositAmount |string |過去30天的入金金額 |
|takerVolTotal |string |過去一年的吃單交易量 |
|makerVolTotal |string |過去一年的掛單交易量 |
|tradeVolTotal |string |過去一年的總交易量 |
|depositAmountTotal |string |過去一年的總入金金額 |
|totalWalletBalance |string |資產餘額 |
|depositUpdateTime |string |上一次入金時間 |
