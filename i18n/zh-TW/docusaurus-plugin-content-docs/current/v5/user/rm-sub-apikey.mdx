---
title: 刪除子帳戶下的API Key
sidebar_label: 刪除子帳戶下的API Key
sidebar_position: 9
---

刪除子帳戶下的api key。使用待刪除的api key調用接口。需使用<b>子</b>帳戶的API key。

:::tip
在調用接口時，使用的API key至少需要擁有以下其中一種權限
* 子API key: "Account Transfer（資產帳戶劃轉）"
:::

:::danger
當心! 用於調用本接口的API KEY將會立馬失效。
:::

### HTTP 請求
POST `/v5/user/delete-sub-api`

### 請求參數
無


### 返回參數
無


### 請求示例

```http
POST /v5/user/delete-sub-api HTTP/1.1
Host: api.bybit.com
X-BAPI-API-KEY: XXXXXX
X-BAPI-TIMESTAMP: 1676431922953
X-BAPI-RECV-WINDOW: 5000
X-BAPI-SIGN: XXXXXX
Content-Type: application/json

{

}
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "",
```
