設置槓桿
> **統一帳戶覆蓋範圍: USDT永續 / USDC永續 / USDC交割 / 反向合約**<br/>
> **經典帳戶覆蓋範圍: USDT永續 / 反向合約**

### HTTP 請求
POST `/v5/position/set-leverage`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |產品類型 <ul><li>統一帳戶: `linear`, `inverse`</li><li>經典帳戶: `linear`, `inverse`. 這裡`category`字段不參與業務邏輯，僅做路由使用</li></ul> |
|symbol |**true** |string |合約名稱|
|buyLeverage |**true** |string |[`1`, [風險限額](../market/risk-limit#)允許的最大槓桿數] <ul><li>經典帳戶: 單倉模式下, `buyLeverage` 必須等於`sellLeverage`</li><li>統一帳戶: 無論什麼模式, `buyLeverage`必須等於`sellLeverage`</li></ul> |
|sellLeverage |**true** |string |[`1`, [風險限額](../market/risk-limit#)允許的最大槓桿數] <ul><li>經典帳戶: 單倉模式下, `buyLeverage` 必須等於`sellLeverage`</li><li>統一帳戶: 無論什麼模式, `buyLeverage`必須等於`sellLeverage`</li></ul>|

<TabItem value="python" label="Python">

```python
from pybit.unified_trading import HTTP
session = HTTP(
    testnet=True,
    api_key="XXXXX",
    api_secret="XXXXX",
)
print(session.set_leverage(
    category="linear",
    symbol="BTCUSDT",
    buyLeverage="6",
    sellLeverage="6",
))
```

</TabItem>
<TabItem value="Node" label="Node.js">

```n4js
const { RestClientV5 } = require('bybit-api');

const client = new RestClientV5({
    testnet: true,
    key: 'apikey',
    secret: 'apisecret',
});

client
    .setLeverage({
        category: 'linear',
        symbol: 'BTCUSDT',
        buyLeverage: '6',
        sellLeverage: '6',
    })
    .then((response) => {
        console.log(response);
    })
    .catch((error) => {
        console.error(error);
    });
```
