通過訂閱DCP流來觸發功能

**舉例:** 私有連接A訂閱了"dcp", 而私有連接B沒有. 如果連接A仍然在線, 但是連接B斷線了, 那麼這種情況下不會觸發DCP功能. 然而, 如果連接A斷線了, 但是連接B仍然在線, 當達到時間窗口閾值時, DCP將會觸發

**Topic:** `dcp`

### 訂閱示例
```json
{
    "op": "subscribe",
    "args": [
        "dcp"
    ]
}
```
